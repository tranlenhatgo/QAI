# TC-02: Quiz Play

## Preconditions

- User is logged in
- All services running (including AI Coach for generated questions)

## TC-02-01: Play a Generated Quiz from Home

**Steps:**

1. On the Home page, click a **category card** (e.g., `Math`)
2. Verify the Play modal/form opens with category pre-selected
3. Confirm `Math` is shown as the selected category (radio filled)
4. Set question count (e.g., 5 via slider/input)
5. Select game mode: **Classic** (no timer)
6. Click the **Start/Play** button
7. Verify questions load and first question displays with 4 answer options

**Expected:**

- Category pre-filled from the clicked card
- Questions generated by AI Coach (may take a few seconds)
- Play page shows question text + 4 clickable answer buttons
- Category-colored background visible

## TC-02-02: Answer Questions and Complete Quiz

**Steps:**

1. (Continuing from TC-02-01 on the play page)
2. Click an answer option for the first question
3. Observe visual feedback (green = correct, red = wrong)
4. Wait for auto-advance or click next arrow/button
5. Answer all remaining questions
6. Verify Game Over screen appears after last question

**Expected:**

- Correct answer: green highlight, score increments
- Wrong answer: red highlight, correct answer revealed
- After last question: GameOver screen with score (e.g., `3/5`)
- Confetti animation if score > 50%

## TC-02-03: Play Existing Quiz via Quiz Browser

**Steps:**

1. Click **logo** to return Home
2. Click the **"Play"** button to open game configuration
3. Look for a quiz browser/list or "Join Quiz" option
4. Find `[AUTOTEST] Math Basics Quiz` in the quiz list
5. Select it and start the quiz
6. Answer the question (`What is 2 + 2?` → click `4`)
7. Complete the quiz

**Expected:**

- Existing quiz loaded from Spring Boot (not AI-generated)
- Questions match what was created in TC-01
- Score saved to quiz history
- Appears in Profile → attempt history

## TC-02-04: Timer Mode

**Steps:**

1. Return to Home (click logo)
2. Click the **"Play"** button
3. Select a category and set **Timer mode** (e.g., 15s)
4. Click Start
5. Wait on one question without clicking any answer
6. Observe timer countdown and auto-advance

**Expected:**

- SVG circular timer visible and counting down
- Timer pulses red when < 6 seconds
- After timer expires: auto-advances (counts as wrong)

## TC-02-05: Wildcards (50/50, Skip)

**Steps:**

1. Start a **Classic** game (wildcards available)
2. On the first question, click the **50/50** wildcard button (top-right area)
3. Observe 2 answers disappear/become disabled
4. Click the **Skip** wildcard button
5. Observe question skipped to next

**Expected:**

- 50/50: exactly 2 incorrect answers removed
- Skip: advances to next question, no penalty
- Wildcard counter decrements after use
- Disabled/greyed out when exhausted
